## :wrench:	Installation <a name="installation"></a>
1. Clone this repo with

```console
git clone https://github.com/rodrigo4635/dedetisys.git
```

2. Open the folder and run

```console
cd dedetisys && npm i
```

3. To start the project
```console
npm start
```

4. To deploy
```console
npm run build
```
